## Description
Claude Skill to format and export Markdown files, perfect for people using Obsidian for their 
Intenship Journals.

#### How does it work?
It takes the information from any Markdown file and styles it to a modern and professional, but creative
layout. It also automatically exports it to a PDF file ready to share.

## Installation
Anyone can use it by cloning the repo and running Claude Code from inside it:
```
  git clone https://github.com/pascazzioIsAbyayalan/claude-skills.git
  cd claude-skills
  claude
```
## Recomendations
- Point out to Claude to modify the issue according where in your Vault (Obsidian), or other .md compatible
  file manager, you have store your report
- Also provide Claude with a location where to store your new PDF docs
- Feel free to ask Claude to modify colour palettes and have fun with it!
- Tweak it to your preference 👾
